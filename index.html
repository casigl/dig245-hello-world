Learn Computing Activity

  • In this activity, we focused on learning about the command line. The command line allows users to communicate with the computer's operating system via text commands. We started practicing with the command whoami, which returns the username of the current user, i.e. carolinesigl. Then, we tried pwd to show the path of the current directory and ls which lists all the files within the current directory. Next, we created a file called hello.txt and viewed it within our current directory using the touch and ls commands. Then, we added text into our file using echo and viewed our edits using cat. We then pinged the Davidson College website which allowed us to see the package communication speeds. Lastly, we used the ls -l command to see the dates of modification on our files and open . to view our current directory in Finder.
